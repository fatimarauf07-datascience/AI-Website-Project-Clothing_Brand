# AI Website Project – Clothing Brand Website

## Project Title

**AI-Powered Clothing Brand Website**

---

## Project Description

This project is a modern and responsive clothing brand website created with the help of AI website generation tools. The website is designed to showcase fashion products with an attractive user interface, responsive layout, animations, and user-friendly navigation.

It demonstrates how artificial intelligence can assist in building professional websites efficiently.

---

## AI Tool Used

**ChatGPT / Cloud AI**

---

## Prompt Used

Create a professional clothing brand website with:

* Homepage
* About Us page
* Products section
* Contact page
* Responsive design
* Smooth animations
* Soft color theme
* Attractive UI
* Call-to-action buttons
* Modern and clean layout

---
